Detailed Description

The methods of this class are used for preparing the render area, viewport, etc. It wraps the actual rendering code used (DirectX, OpenGL...)

bool IND_Render::reset ( IND_WindowProperties &  props)

Parameters:

  • pTitle Title of the window
  • pWidth Width of the window
  • pHeight Height of the window
  • pBpp Quality of color (32 o 16 bits). 32 bits offers better quality. 16 bits offers better speed.
  • pVsync Wait for vertical sync. (1 / 0) = (on / off).
  • pFullscreen Full screen. (1 / 0) = (on / off).

Operation:

This function returns 1 (true) if the application window resets to the attributes passed as parameters. This method is useful when you want to change the application window on runtime, 0 (false) if it is not possible to create the new window.

Operation:
