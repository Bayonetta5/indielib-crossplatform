Detailed Description

This class is used when loading RLE compressed images, it implements io cache of fixed size. In general RLE compressed images should be compressed line by line with line sizes stored in Scan Line Table section. In reality, however there are images not obeying the specification, compressing image data continuously across lines, making it impossible to load the file cached at every line.
