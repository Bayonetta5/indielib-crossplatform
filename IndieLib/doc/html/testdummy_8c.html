Detailed Description

Dummy test suite for test runner. This can be used as a base for writing new tests. Dummy suite also works as reference to using various asserts and other (possible) utilities.

Function Documentation

void SetUp ( void *  arg)

SetUp function can be used to create a test fixture for test cases. The function will be called right before executing the test case.

Note: If any assert in the function fails then the test will be skipped. In practice, the entire suite will be skipped if assert failure happens.

Note: this function is optional.

argparameters given to test. Usually NULL
